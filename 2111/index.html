2111/9111 STATIC RAM REPLACEMENT -- PCB not tested yet! --

This project is about replacing a single faulty 2111/9111 RAM with a more recent 6116/62256.
The pin strips to the 2111 to be replaced must extend out from the soldering side, this PCB is on top.
The RAM model is selected by shorting two pads on the soldering side.

Note: C1=100 nF, R1..R4=10 kohm

POLYMORPHIC VIDEO TERMINAL INTERFACE (VTI) BULK RAM REPLACEMENT -- PCB not tested yet! --

This project is about replacing all the 2111/9111 RAM ICs on the POLYMORPHIC "Video Terminal Interface" (VTI) board with a single 6116/62256.

Link to VTI Documentation

All RAM IC's must be removed. The board have to be placed on top of the sockets IC23/IC27 and connected with appropriate pin strips.
The pin strips to the VTI's RAM sockets must extend out from the soldering side.
RAM Model is selected by the jumper.
