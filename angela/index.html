APPLE-1 "ANGELA GAME" REFERENCE PAGE


Porting of "ANGELA GAME", originally developed for Olivetti Programma 101 Desktop Computer by:

- Pier Giorgio Perotto
- Gastone Garziera
- Giovanni De Sandre

shown for the first time at B.E.M.A (Business Equipment Manufacturers Association) in October 1965

This porting runs on Applesoft BASIC (not INTEGER Basic), available with the Apple-1 microSD Storage Card.

The binary version of the game has already been integrated into the SD card's program archive.
