APPLE-1 "DOBBLE" REFERENCE PAGE


Porting of card game "DOBBLE", originally devised by french mathematics enthusiast Jacques Cottereau.

Inspired by the game "Dobble" written for Commodore 64 by

- Jader Monari
- Prof. Massimo Ferri of the University of Bologna (Italy)


Totally rewritten in Applesoft BASIC for Apple-1.
Also ported to INTEGER BASIC and shrinked down to <4kB to be run on unexpanded Apple-1/Replica.


The binary version of the game has already been integrated into the SD card's program archive.

APPLE-1 "THE PAPER CUP MACHINE" AKA "LA MACCHINA A BICCHIERI" REFERENCE PAGE


Porting of "La Macchina a Bicchieri (MAB)", originally written for Commodore C64 by:

- Jader Monari

Devised from an idea of:

- Prof. Massimo Ferri of the University of Bologna (Italy)

Also based on the MENACE Matchbox Machine designed and built by researcher Donald Michie in 1961:

- https://en.wikipedia.org/wiki/Matchbox_Educable_Noughts_and_Crosses_Engine

Ported to Applesoft BASIC thanks to the efforts of:

- Francesco Sblendorio

This porting runs on Applesoft BASIC (not INTEGER BASIC), available with the Apple-1 microSD Storage Card.

The two binary versions of the game have already been integrated into the SD card's program archive:
- MAB-ITA for the Italian version
- PCM-ENG for the English version
